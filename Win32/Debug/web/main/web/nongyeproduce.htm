TLINK物联网平台——农业大棚监测解决方案
       设施农业是采用人工技术手段，改变自然光照条件，创造优化动植物生长的环境因子，使之能够全天候生长的设施工程，多用于低温季节喜温蔬菜、花卉、水果等植物栽培或育苗等。通过物联网技术，真正实现农业生产自动化、管理智能化，使设施种植管理智能化调温、精准化施肥，可达到提高产量、高品质、省人力、降成本、提高经济效益的目的。【该方案由合作企业无锡卓感科技有限公司提供】

    方案概述 传感器 控制层 通信层 平台层
传感层
通过部署的无线传感器节点和病虫害监测设备以及图像采集设备，对虫情、墒情、灾情、墒情进行实时监测，同时对信息的实时传送的通信模块和网络的部署，传感层是实现智能灌溉、通风、卷帘的前提。
控制层
控制层在现代农业自动控制中运用物联网系统传感器设备，检测农业环境要素，自动控制温室湿帘风机、喷淋滴灌、内外遮阳、顶窗侧窗、加温补光等设备，达到适宜植物生长的范围，为植物生长提供最佳环境。
通信层
农业物联网需要从云端到终端的整体解决方案 ;通信层目的是实现本地和云端的数据同步;
由本地服务器进行智能化处理后提取特征数据传回云端
4G工业路由器是基于无线通信的工业级路由器，利用公用无线网络为设备提供无线数据传输功能，传输速率更快更稳定，更能适应恶劣环境应用
平台层
远程获取大田或者温室大棚内部的空气温湿度/土壤水分温度/二氧化碳浓度/光照强度及视频图像;通过模型分析，自动控制温室湿帘风机/喷淋滴灌/内外遮阳/顶窗侧窗/加温补光等设备。同时，系统还可以通过手机/PAD/计算机等终端向管理者推送实时监测信息、报警信息，实现温室大棚信息化、智能化远程管理，充分发挥物联网技术在设施农业生产中的作用保证温室大棚内环境最适宜作物生长实现精细化的管理,为作物的高产、优质、高效、生态、安全创造条件， 提高效率、降低成本、增加收益。
