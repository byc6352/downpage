TLINK物联网平台——工业制造监测解决方案
       本案例讲述采集生产线各个工段的生产工艺数据，并集中远传云端实现远程监控，所用到的传感器有温度传感器、压力传感器、液位传感器，传感器的数据统一传给工业主机，工业主机把处理好的数据传给云端。生产员工可以实时得到现场主机的报警反馈，管理人员也可以随时随地监测生产参数。对于工厂监控，本地着重看数据的实时变化，并能及时报警提示，云端着重看历史趋势。根据 每天每月数据做对比，来对生产工艺做改进。目前的方案可以实现对生产的产品品质进行管控，使产品合格率得到提升。【该方案由合作企业深圳市拓普瑞电子有限公司提供】

    方案概述 传感器 设备层 通信层 平台层
传感层
传感器工厂生产现场需要安装大量的传感器来感知每个工序的生产参数,PLC采集传感器并进行判断给出反馈输出PLC再和其他的仪表、电表、HMI一同接入SCADA系统,实现数据的本地汇总和
设备层
PLC采集传感器并进行判断给出反馈输出；HMI实现人机交互、逻辑计算、报警、控制；SCADA软件实现数据的本地监控、汇总、计算、分析及时响应设备数据请求和控制设备的行为；有效解决工业数据调度的一致性与完整性问题
通信层
工业物联网需要从云端到终端的整体解决方案
通信层目的是实现本地和云端的数据同步
由SCADA进行智能化处理后提取特征数据传回云端
工业网关分为透传（2G/3G/4G DTU）和非透传(集成多种协议的嵌入式盒子)两种
平台层
平台提供设备连接能力和相应的应用功能；设备连接支持多种连接协议（TCP、ModbusRTU、ModbusTCP、MQTT）；
触发器支持多种触发条件和微信、短信、邮件三种报警方式；
组态应用个性化定义传感器数值、开关、图表、曲线。
